Arrow Functions

Let's learn how to create arrow functions.

They're part of the new ES6 JavaScript standard and are used to write cleaner and shorter code.

Take a look at how we're rewritten the function below as an arrow function in just one line of code.

function getGreeting() {
return 'Hello';
};

var greeting = () => 'Hi';

console.log(getGreeting());
console.log(greeting());
